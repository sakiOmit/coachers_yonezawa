# トップページ設定手順

## 1. WordPress管理画面にログイン

```
URL: http://localhost:8000/wp-admin/
```

## 2. 固定ページを新規作成

### 手順

1. **固定ページ > 新規追加** をクリック
2. 以下の内容を入力:

   - **タイトル**: `トップページ` または `ホーム`
   - **パーマリンク**: `home` または `top`（URLスラッグ）

3. **ページ属性** セクションで:
   - **テンプレート**: `トップページ` を選択
   - このテンプレートが表示されない場合は、テーマが正しくアップロードされているか確認してください

4. **公開** ボタンをクリック

## 3. ACFフィールドを設定（自動表示）

ページテンプレートを「トップページ」に設定すると、自動的に以下のACFフィールドが表示されます:

### トップページ設定

- **メインビジュアル タイトル**
  - デフォルト: `Making Fans Of Japan`
  - 変更したい場合は任意のタイトルを入力

- **メインビジュアル サブタイトル**
  - デフォルト: `「JAPAN QUALITY」を世界へ発信し、日本のファンをつくる`
  - 変更したい場合は任意のサブタイトルを入力

空白のままでもデフォルト値が表示されます。

## 4. フロントページに設定

### 手順

1. **設定 > 表示設定** に移動
2. **ホームページの表示** セクションで:
   - 「固定ページ」を選択
   - **ホームページ**: 作成した「トップページ」を選択
3. **変更を保存** をクリック

## 5. 動作確認

以下のURLにアクセスして確認:

```
http://localhost:8000/
```

### 確認項目

- ✅ ヘッダーが表示される（ロゴ、ナビゲーション、CTAボタン）
- ✅ メインビジュアルが表示される（タイトル、サブタイトル）
- ✅ フッターが表示される（ロゴ、ナビゲーション、会社情報）
- ✅ レスポンシブ対応（スマホ表示でハンバーガーメニューが表示）

## トラブルシューティング

### テンプレートが表示されない

```bash
# テーマが正しい場所にあるか確認
ls -la themes/{{THEME_NAME}}/pages/

# page-top.php が存在することを確認
cat themes/{{THEME_NAME}}/pages/page-top.php
```

### ACFフィールドが表示されない

1. **プラグイン > Advanced Custom Fields** がインストール・有効化されているか確認
2. **カスタムフィールド > フィールドグループ** で「トップページ設定」が存在するか確認
3. `themes/{{THEME_NAME}}/acf-json/group_top_page.json` が存在するか確認

### スタイルが反映されない

```bash
# ビルドを再実行
npm run build

# または開発モードで確認
npm run dev
```

### パーマリンク404エラー

**設定 > パーマリンク設定** を開いて「変更を保存」をクリック（設定を変更しなくても保存するだけでOK）

## 次のステップ

- ハンバーガーメニューのJavaScript実装
- トップページのコンテンツ追加（フレキシブルコンテンツ対応予定）
