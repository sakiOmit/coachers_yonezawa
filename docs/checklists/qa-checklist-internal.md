# 社内品質保証チェックシート

社内ダブルチェック用の品質保証チェックシートです。自動チェック可能な項目と手動確認が必要な項目を明確に区分しています。

## 使い方

### 自動チェック

Claude Codeの `/delivery-check` コマンドで自動チェックを実行できます。

```bash
# 全チェック実行
/delivery-check

# 特定カテゴリのみ
/delivery-check --category=technical
/delivery-check --category=seo
```

### 手動チェック

自動チェックでカバーできない項目は手動で確認し、このシートにチェックを入れてください。
