# Agents Rules

## Overview

このルールファイルは、エージェント定義およびMCP参照に関する規約を定義します。

## MCP参照規約

エージェント定義でMCPツールを参照する際は、以下を確認せよ:

1. **導入確認**: .claude/mcp-configs/ に該当MCPが登録されているか
2. **ツール名確認**: 正確なツール名を使用（mcp__figma__ 等）
3. **未導入の場合**: 「将来対応予定」と明記するか、セクションを削除

### 禁止事項

| 禁止項目 | 理由 |
|---------|------|
| 存在しないMCPツールの参照 | 動作不能の原因 |
| 古いツール名の使用 | リネーム後の旧名参照 |

## チェックリスト

- [ ] MCPツール参照は導入済みのもののみ
- [ ] ツール名は正確（mcp__{server}__{tool}形式）
- [ ] 未導入MCPは明記または削除
