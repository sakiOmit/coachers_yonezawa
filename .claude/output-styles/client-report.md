# Client Report Output Style

クライアント提出用レポートの出力形式を定義します。

## 適用場面

- `delivery-checker` エージェントの出力
- `/delivery-check` コマンドの結果
- 納品時の品質報告書

## 出力構成

### 1. ヘッダー

```markdown
# 品質保証レポート

プロジェクト名: [プロジェクト名]
作成日: YYYY年MM月DD日
作成者: [会社名/担当者名]
```

### 2. エグゼクティブサマリー

```markdown
## 概要

本レポートは、Webサイト納品前の品質チェック結果をまとめたものです。

### 総合評価: ✅ 納品準備完了

すべての主要項目で品質基準をクリアしています。
```

### 3. チェック結果一覧

```markdown
## 品質チェック結果

| カテゴリ | 項目 | 状態 | 備考 |
|---------|------|:----:|------|
| 表示 | PC表示 | ✅ | Chrome, Edge で確認済み |
| 表示 | スマートフォン表示 | ✅ | iOS Safari, Android Chrome で確認 |
| 表示 | タブレット表示 | ✅ | iPad で確認済み |
| 機能 | フォーム送信 | ✅ | メール送信テスト完了 |
| 機能 | ページ遷移 | ✅ | 全リンク動作確認済み |
| SEO | メタ情報設定 | ✅ | 全ページ設定済み |
| SEO | OGP設定 | ✅ | SNS共有テスト完了 |
| 性能 | 表示速度 | ✅ | 目標値クリア |
| 性能 | 画像最適化 | ✅ | WebP変換済み |
| アクセシビリティ | 代替テキスト | ✅ | 全画像設定済み |
| セキュリティ | SSL対応 | ✅ | HTTPS強制リダイレクト設定済み |
```

### 4. 詳細（必要な場合）

```markdown
## 詳細情報

### 表示確認

**対応ブラウザ:**
- Google Chrome（最新版）
- Microsoft Edge（最新版）
- Safari（最新版）
- Firefox（最新版）

**対応デバイス:**
- デスクトップ: 1920px, 1440px, 1280px
- タブレット: 768px
- スマートフォン: 375px, 390px

### パフォーマンス測定結果

| 指標 | 結果 | 基準値 |
|------|------|--------|
| First Contentful Paint | 1.2秒 | 2.0秒以下 |
| Largest Contentful Paint | 2.1秒 | 2.5秒以下 |
| Time to Interactive | 2.8秒 | 3.0秒以下 |
```

### 5. 注意事項（必要な場合）

```markdown
## 注意事項

以下の項目について、運用開始後にご確認をお願いいたします：

1. **お問い合わせフォーム**
   - 実際のメールアドレスへの送信テストをお願いします
   - 自動返信メールの内容をご確認ください

2. **Google Analytics**
   - データ計測開始後、動作確認をお願いします
```

### 6. フッター

```markdown
---

ご不明な点がございましたら、お気軽にお問い合わせください。

[会社名]
[担当者名]
[連絡先]
```

## フォーマットルール

### 言語・表現

- **敬語**を使用
- **技術用語を避ける**（または括弧内で補足説明）
- **ポジティブな表現**を優先

```markdown
✅ 良い例: 「品質基準をクリアしています」
❌ 避ける: 「エラーは検出されませんでした」

✅ 良い例: 「画像の表示速度を最適化しました」
❌ 避ける: 「WebP変換とLazy Loadingを実装」
```

### 状態表示

| 記号 | 意味 | 使用場面 |
|:----:|------|---------|
| ✅ | 完了/問題なし | 基準クリア時 |
| ⚠️ | 確認推奨 | 軽微な注意事項 |
| ❌ | 未完了/要対応 | 重大な問題（通常は納品前に解決） |

### 数値の表記

- パーセント: `95%` ではなく `約95パーセント`
- 時間: `1.2s` ではなく `1.2秒`
- ファイルサイズ: `500KB` ではなく `約500キロバイト`

### 日付表記

- `YYYY年MM月DD日` 形式を使用
- 例: `2025年12月12日`
